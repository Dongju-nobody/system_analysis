# Quickstart — My First Wiki Page

Clone 후 **Browse** 탭에서 이 페이지를 확인할 수 있습니다.

## 수정 방법

1. 이 파일을 복사해 `wiki/notes/my-topic.md` 생성
2. `title`과 본문을 본인 자료로 교체
3. `npm run wiki:dev` 실행 후 Browse → **notes** 섹션에서 선택

## 맥락

Package 30분 Quickstart 2단계: **내 자료 1건 추가** 예시.
