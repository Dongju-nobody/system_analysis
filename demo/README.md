| 파일 | 설명 |

## 직접 캡처 방법

1. `npm run wiki:dev`
2. Browse → notes → `quickstart-my-first-page.md`
3. Parties 탭에서 파티 1건 선택
4. Win+Shift+S 로 캡처 후 이 파일 교체

민감 내용은 가리고 UI만 보여도 됩니다.
